# 🎯 Быстрый старт: Кураторские темы

## 🚀 За 60 секунд

### 1. Откройте приложение
```
http://localhost:5173/topics
```

### 2. Выберите категорию
```
🪖 Война / Безопасность
🌍 Экология / Климат
🏛️ Политика / Общество
💡 Технологии / Экономика
```

### 3. Кликните на тему или "Искать"
```
Одна тема → кнопка "Искать" → 20 видео
Вся категория → кнопка "Искать все" → 100+ видео
```

### 4. Просматривайте результаты
```
Таблица → Сортировка → Клик на видео → YouTube открывается
```
