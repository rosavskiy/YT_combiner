# 🎬 Работа со ссылками на YouTube

## ✨ Что добавлено

Таблица трендов теперь **интерактивная** - каждое видео и канал это кликабельная ссылка!

## 🖱️ Как использовать

### 1. Открыть видео (3 способа)

#### Способ 1: Кликнуть на название
```
┌──────────────────────────────────────┐
│ 🔴 Название видео (синяя ссылка)     │ ← Клик
└──────────────────────────────────────┘
        ↓
   Откроется YouTube
```

#### Способ 2: Кликнуть на канал
```
┌──────────────────────┐
│ Название канала       │ ← Клик (серая ссылка)
└──────────────────────┘
        ↓
   Страница канала YouTube
```

#### Способ 3: Кнопка "Открыть"
```
┌─────────────┐
│ 🔗 Открыть  │ ← Клик (справа в таблице)
└─────────────┘
        ↓
   Откроется YouTube
```

## 🎯 Что происходит при клике

### Клик на видео:
1. Открывается **новая вкладка браузера**
2. Загружается страница: `https://www.youtube.com/watch?v=VIDEO_ID`
3. Видео начинает воспроизведение
4. Текущая вкладка (приложение) остается открытой

### Клик на канал:
1. Открывается **новая вкладка**
2. Загружается: `https://www.youtube.com/channel/CHANNEL_ID`
3. Показывается страница канала со всеми видео

```

2. Отсортировать по просмотрам (клик на колонку)

4. Просмотреть за раз
```

## 🔒 Безопасность

Все ссылки открываются **безопасно**:
- ✅ `target="_blank"` - новая вкладка
- ✅ `rel="noopener"` - защита от tabnabbing
- ✅ `rel="noreferrer"` - не передает referrer

## 🎨 Визуальные индикаторы

| Элемент | Цвет | Иконка | Действие |
|---------|------|--------|----------|
| Название видео | 🔵 Синий | 🔴 YouTube | Открыть видео |
| Канал | ⚫ Серый | - | Открыть канал |
| Кнопка "Открыть" | 🔵 Синий | 🔗 Link | Открыть видео |

## 📱 Адаптивность

Таблица адаптируется под размер экрана:
- **Desktop**: Все колонки видны
- **Tablet**: Горизонтальный скролл
- **Mobile**: Фиксированы "Страна" и "Действия"

## ⌨️ Горячие клавиши

```
Ctrl + Click   = Открыть в фоновой вкладке
Shift + Click  = Открыть в новом окне
Middle Click   = Открыть в новой вкладке (на мышке)
```
